# 插件接口

包名：com.codecheck.main.api; 接口名：CodeCheckConfig;
以下方法是接口中的核心方法，其他方法就不再列举了。

接口方法 方法参数 返回值 方法说明
String provideHtmlSetting(); 返回值为html的基础样式，div标签内的内容 提供在html页面可以进行检查的检查项
String getHtmlSetting(String content); content为配置后的json字符串 返回值为设置过的html字符串，div标签内的包含参数值的字符串 获取在html页面所配置的检查项信息，进行配置后返回html中的div
String provideConfiguredXMLRules(String content); content为配置后的json字符串 返回值为配置后的checkstyle检查规则字符串 提供已配置好的checkstyle.xml文件
String getScoreSetting(String content); content为配置后的json字符串 返回值为分数设置文件的字符串内容 获取配置后的分数设置

包名：com.codecheck.main.api; 接口名：CodeCheckAnalyse; 以下方法是接口中的核心方法。

接口方法 方法参数 返回值 方法说明
String createChartAnalysis(String projectPath); projectPath为工程路径名 返回值为图表的路径（projectPath+/target/charts/codecheck） 对代码检查结果创建图表
String getCheckScore(String projectPath); projectPath为工程路径名 返回值为检查过后的分数字符串 获取代码检查得分
String getCheckDetail(String projectPath); projectPath为工程路径名 返回值为代码检查的具体的信息 获取代码检查的详细信息

# 接口实现类

上述接口的实现类分别为：
包名：com.codecheck.main.impl; 实现类名：CodeCheckConfigImpl;
对应的实现类中的方法和接口中定义的一样，在此不在赘述。

包名：com.codecheck.main.impl; 实现类名：CodeCheckAnalyseImpl;
对应的实现类中的方法和接口中定义的一样，在此不在赘述。

CodeCheckAnalyseImpl举例如下：

public class CodeCheckAnalyseImplTest {

	CodeCheckAnalyseImpl codeCheckAnalysis;
	String projectPath;  //构建后的工程路径名
	
	@BeforeClass
	public static void setUpBeforeClass() throws Exception {
	}

	@AfterClass
	public static void tearDownAfterClass() throws Exception {
	}

	@Before
	public void setUp() throws Exception {
		codeCheckAnalysis=new CodeCheckAnalyseImpl();
		projectPath="D:/testplugin/sort";
	}

	@After
	public void tearDown() throws Exception {
	}

	@Test
	public void testCreateChartAnalysis() {
		//调用插件的生成图表方法，返回生成图表所在的路径
		String path=codeCheckAnalysis.createChartAnalysis(projectPath);
		System.out.println(path);
	}

	@Test
	public void testGetCheckScore() {
		//调用插件的获取检查分数方法，返回分析过后静态代码检查的得分
		String result=codeCheckAnalysis.getCheckScore(projectPath);
		System.out.println(result);
	}

	@Test
	public void testGetCheckDetail() {
		//调用此方法，获取静态代码检查的详细信息，以字符串的形式返回
		String result=codeCheckAnalysis.getCheckDetail(projectPath);
		System.out.println(result);
	}
}
