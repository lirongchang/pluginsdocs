# 使用介绍

# 插件简介

本次共开发了以下7款插件，主要用于自动化软件测试课程的教学，使用者通过调用插件接口，来实现对学生提交的作业进行检查。
进行检查的插件主要是静态代码规范性检查插件，单元测试通过率检查插件，单元测试覆盖率检查插件，代码查重插件。
进行分析的插件则是作业分数分析插件和汇总分析插件。
进行构建的插件则是Maven构建插件。

# 动态配置

在本次开发的插件中，动态配置需要由调用者将插件提供的html页面中的div标签内容嵌入到自己的web页面中，以实现通过web前端对插件进行动态配置。

# mavenbuild

用于对学生提交的maven工程进行构建。

# codecheck

用于静态代码规范性检查。

# passingcheck

用于单元测试通过率检查。

# coveragecheck

用于单元测试覆盖率检查。

# scoringanalysis

基于静态代码检查、单元测试通过率检查和单元测试覆盖率检查，用于分析学生作业成绩。

# codeduplicate

用于检查学生提交的作业是否存在抄袭。

# summaryanalysis

用于对上述其他插件的检查分析数据，按班级进行汇总分析。代码查重插件则按整体进行的分析。
